Sintaxis

Listas

Listas ordenadas: Son las que ordenan la lista anteponiendo números, letras o signos
en la cual comienzan con la etiqueta ol y para cada uno de los elementos se utiliza
la etiqueta li.

Listas desordenadas: Son las que muestran la lista anteponiendo un punto, cuadrado o triángulo
a cada elemento, comienzan con la etiqueta ul o dir y para cada uno de los elementos de la
se emplea la etiqueta li.
